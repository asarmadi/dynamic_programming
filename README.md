# Dynamic Programming
In this repository, dynamic programming is implemented to solve various problems.

## Install
A python virtual environment could be generated

`
python -m venv env
`

and activates in windows

`
.\env\Scripts\activate
`

in Linux

`
source ./env/bin/activate
`
